### Import segmentation links

Import the connection between the main project and this segmentation from a table.
Each row in the table represents a connection between one base vertex and one segment.

====
include::{g}[tag=table-import]

[[base-id-attr]] Identifying vertex attribute in base project::
The `String` vertex attribute that can be joined to the identifying column in the table.

[[base-id-column]] Identifying column for base project::
The table column that can be joined to the identifying attribute on the base project.

[[seg-id-attr]] Identifying vertex attribute in segmentation::
The `String` vertex attribute that can be joined to the identifying column in the table.

[[seg-id-column]] Identifying column for segmentation::
The table column that can be joined to the identifying attribute on the segmentation.
====
