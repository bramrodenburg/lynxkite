## Import vertex attributes from a database

Imports vertex attributes for existing vertices from a SQL database.

====
include::glossary.asciidoc[tag=database-import]

[[id-attr]] Vertex ID attribute::
The vertex attribute which is used in the database's ID column.

[[id-field]] ID field::
The ID column name in the database. This should be a string column that uses the same IDs as
the chosen vertex ID attribute.

[[prefix]] Name prefix for the imported vertex attributes::
Prepend this prefix string to the new vertex attribute names. This can be used to avoid
accidentally overwriting existing attributes.
====
