### Import segmentation from CSV

Imports a segmentation from a CSV. The CSV must have a field identifying an existing vertex by
a string attribute and another field that specifies the segment it belongs to.
Each vertex may belong to any number of segments.

The rest of the fields in the CSV are ignored.

====

The CSV field that contains vertex IDs.

[[seg-id-field]] Segment ID field::
