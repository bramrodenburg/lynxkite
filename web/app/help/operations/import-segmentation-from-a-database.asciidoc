### Import segmentation from a database

Imports a segmentation from a SQL database. The database table must have a column identifying an
existing vertex by a string attribute and another column that specifies the segment it belongs to.
Each vertex may belong to any number of segments.

The rest of the columns in the table are ignored.

====
