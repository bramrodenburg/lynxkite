### Import edge attributes from a database

Imports edge attributes for existing edges from a SQL database.

====
include::{g}[tag=database-import]

[[id-attr]] Edge ID attribute::
The edge attribute which is used in the database's ID column.

[[id-field]] ID field::
The ID column name in the database. This should be a string column that uses the same IDs as
the chosen edge ID attribute.

[[prefix]] Name prefix for the imported edge attributes::
Prepend this prefix string to the new edge attribute names. This can be used to avoid
accidentally overwriting existing attributes.
====
