### Import edge attributes

Imports edge attributes for existing edges from a table. This is
useful when you already have edges and just want to import one or more attributes.

====
include::{g}[tag=table-import]

[[id-attr]] Edge ID attribute::
The edge attribute which is used in the table's ID column.

[[id-column]] ID column::
The ID column name in the table. This should be a string column that uses the same IDs as
the chosen edge ID attribute.

[[prefix]] Name prefix for the imported edge attributes::
Prepend this prefix string to the new edge attribute names. This can be used to avoid
accidentally overwriting existing attributes.

[[unique-keys]] Unique keys::
Assert that the edge ID attribute values have to be unique if set true. The values of the
matching ID column in the table have to be unique in both cases.
====
