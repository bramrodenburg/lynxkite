## Load segmentation links from a database

Import the connection between the main project and this segmentation from a SQL database.
Each database row represents a connection between one base vertex and one segment.

====
include::glossary.asciidoc[tag=database-import]

[[base-id-attr]] Identifying vertex attribute in base project::
The `String` vertex attribute that can be joined to the identifying column in the database.

[[base-id-field]] Identifying column for base project::
The database column that can be joined to the identifying attribute on the base project.

[[seg-id-attr]] Identifying vertex attribute in segmentation::
The `String` vertex attribute that can be joined to the identifying column in the database.

[[seg-id-field]] Identifying field for segmentation::
The database column that can be joined to the identifying attribute on the segmentation.
====
