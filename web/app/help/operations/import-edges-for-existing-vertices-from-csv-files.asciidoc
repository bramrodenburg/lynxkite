## Import edges for existing vertices from CSV files

Imports edges from a CSV file, or files. Your vertices must have an identifying attribute, by which
the edges can be attached to them.

====
include::glossary.asciidoc[tag=csv-import]

[[attr]] Vertex ID attribute::
The IDs that are used in the file when defining the edges.

[[src]] Source ID field::
The CSV field that specifies the source of the edge.

[[dst]] Destination ID field::
The CSV field that specifies the destination of the edge.
====
